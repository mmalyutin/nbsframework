Design Reports.

To design a report use special XML file (*.report.xml).
In the report file you can describe following sections:
  • Report Data
  • Report Template
For example: 

<report name="Report1" caption="Report 1">

  <data-sources>
    <data-source>
      <fields>
        <field name="PRODUCT_ID" data-type="Integer"/>
        <field name="PRODUCT_NAME" data-type="String"/>
        <field name="PRICE" data-type="Float"/>
      </fields>
      <query>SELECT PRODUCT_ID, PRODUCT_NAME, PRICE FROM PRODUCT</query>
    </data-source>
  </data-sources>
 
  <templates>
    <template type="Table" cell-border-rule="all" row-line="1" column-line="1">
      <page-setup margin-left="15" margin-top="22" margin-right="23" margin-bottom="24" />
      <columns>
        <column width="100" />
        <column width="150" />
        <column width="150" />
      </columns>
		
      <bands>
        <band type="ReportHeader">
          <rows>
            <row height="30" background="red">
              <cells>
                <cell colspan="4" horizontal-align="center" vertical-align="middle" font="16"><value>My Report</value></cell>
              </cells>
            </row>
          </rows>
        </band>
			
        <band type="ColumnHeader">
          <rows>
            <row height="30">
              <cells>
                <cell font="bold"><value>Product ID</value></cell>
                <cell font="bold"><value>Product Name</value></cell>
                <cell font="bold"><value>Price</value></cell>
              </cells>
            </row>
          </rows>
        </band>
			
        <band type="Detail">
          <rows>
            <row height="20">
              <cells>
                <cell><expression>$F{PRODUCT_ID}</expression></cell>
                <cell><expression>$F{PRODUCT_NAME}</expression></cell>
                <cell format="#.00" data-type="Float"><expression>$F{PRICE}</expression></cell>
              </cells>
            </row>
          </rows>
        </band>
      </bands>
    </template>
  </templates>
  
</report>
In section datasources/datasource we configure the Data Source.
It is SQL Data Source with 3 fields and simple query. All fields have different data types. (PRODUCT_ID - Integer, PRODUCT_NAME - String, PRICE - Float)
In section templates/template we design the Report Template. Type of the Report Template is 'Table'. The table has 3 columns.
In section templates/template/bands we design bands of the Report Template. Bands are special containers of report elements. Each band has own type. For each band we design rows and cells.
