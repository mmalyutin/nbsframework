Types

Type Description Example
String Represents sequence of characters "Hello", "H"
Number Represents numeric values 100, 3.14, -0.0001
Boolean Represents boolean value either false or true false, true
Date Represents date values Date(2001, 12, 31)
List Represents list of elements [1, 2, 3], ["abc", 3.14, 0], List("abc", 3.14, 0)
Map Represents map of pairs (key and value) {name: "Name", amount: 100.12}
