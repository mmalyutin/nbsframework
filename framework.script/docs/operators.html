Arithmetic operators

Operator Description Types Example
+ Addition Number + Number -> Number
String + any -> String
Date + Number -> Date
List + any -> List
 10 + 20 -> 30
"abc" + 10 -> "abc10"
Date(2001, 12, 31) + (24 * 60 * 60 * 1000) -> Date(2002, 1, 1)
[1, 2, 3] + 4 -> [1, 2, 3, 4]
- Subtraction Number - Number -> Number
Date - Number -> Date
List - any -> List
 30 - 20 -> 10
Date(2001, 12, 31) - (24 * 60 * 60 * 1000) -> Date(2001, 12, 30)
[1, 2, 3] - 2 -> [1, 3]
* Multiplication Number * Number -> Number
String * Number -> String
List * Number -> List
 10 * 20 -> 200
"ab" * 3 -> "ababab"
[1, 2] * 4 -> [1, 2, 1, 2, 1, 2, 1, 2]
/ Division Number / Number -> Number 7 / 5 -> 1.4
% Remainder Number % Number -> Number 7 % 5 -> 2
^ Power Number ^ Number -> Number 2 ^ 10 -> 1024
