Control structures

  • Conditional structures
  • Looping structures

Conditional structures

if / else

if-else syntax. 
 if (a >= 0) {
   println("positive");
 } else {
   println("negative");
 }

Looping structures

for in loop

for syntax. 
for (i in 1..10) {
  println("i = " + i);
}

while loop

while syntax. 
var i = 0;	
while (i <= 10) {
  i = i + 1;
  println("i = " + i);
}
