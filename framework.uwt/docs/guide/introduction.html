Introduction.

Plazma UWT is Universal Widget Toolkit.
You can develop your UI widgets in one model and than use it for more UI platforms.

General features:

  • Multi platforms
  • Swap widgets
  • XML design

Multi platforms

UWT supports following UI platforms:
  • Swing
  • SWT
  • Java FX
  • GWT/GXT

Swap widgets

You can replace a widget in one place for your application.

XML design

You can design your UI containers in XML.
