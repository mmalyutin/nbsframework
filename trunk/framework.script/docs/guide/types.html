Types

Type Description Example
String Represents sequence of characters "Hello", "a", ""
Number Represents numeric values 100, 3.14, -0.0001, 0
Boolean Represents boolean value either true or false true, false
Date Represents date values (year, month, day) Date(2001, 12, 31), Date()
Time Represents time values (hour, minute, second, millisecond) Time(18, 30, 25, 150), Time()
DateTime Represents date and time values (year, month, day, hour, minute, second, millisecond) DateTime(2001, 12, 31, 18, 30, 25, 150), DateTime()
List Represents list of elements [1, 2, 3], ["abc", 3.14, 0, true], List("abc", 3.14, 0, true)
Set Represents unique set of elements Set(2, 4, 6, 8, 10), [1, 2, 3, 4, 5].toSet()
Map Represents map of key/value pairs {"name": "Alex", "amount": 5000.00, "paid": true}
