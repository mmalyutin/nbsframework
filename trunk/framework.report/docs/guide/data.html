Report Data.

In the report data section you can configure following elements:
  • Data Connector
  • Data Source
  • Parameters
  • Variables

Data Connector

The Data Connector configures a connect to Data Sources.
It is optional section. We can put a Data Connector in runtime mode.
We have following types of the Data Connector:
  • Database/SQL
  • CSV
  • Json
  • XML
  • Excel (XLS, XLSX)
For example SQL Data Connector: 

<report>

     <data-connectors>
       <data-connector type="sql">
         <properties>
           <property name="driver">org.hsqldb.jdbcDriver</property>         
           <property name="url">jdbc:hsqldb:mem:mydb</property>
           <property name="username">sa</property>
           <property name="password"></property>           
         </properties>
       </data-connector>
     </data-connectors>
	
</report>
For example CSV Data Connector: 

<report>

     <data-connectors>
       <data-connector type="csv">
         <properties>
           <property name="file">../data/Report1.csv</property>
           <property name="firstRowHeader">true</property>
           <property name="rowDelimiter">#LF</property>
           <property name="encoding">UTF-8</property>           
         </properties>
       </data-connector>
     </data-connectors>
	
</report>

Data Source

The Data Source describes flat data structure (like a table)
We have following base elements of the Data Source:
  • Fields
  • Filters
  • Orders
  • Query
For example: 
<report>

  <data-sources>
    <data-source>
    
      <fields>
        <field name="PRODUCT_ID" data-type="Integer"/>
        <field name="PRODUCT_NAME" data-type="String"/>
        <field name="GROUP_NAME" data-type="String"/>
        <field name="PRICE" data-type="Float"/>
      </fields>
      
      <filters>
        <filter field="PRICE" operator="lte">
          <value>150.00</value>
        </filter>
      </filters>
      
      <orders>
        <order field="GROUP_NAME"/>
        <order field="PRODUCT_NAME"/>
      </orders>
      
      <query>SELECT PRODUCT_ID, PRODUCT_NAME, GROUP_NAME, PRICE FROM PRODUCT</query>
    </data-source>
  </data-sources>
  
</report>
